virtualenv 的加强版 virtualenvwrapper

优点：

  1. 将所有虚拟环境整合在一个目录下
  2. 管理（新增，删除，复制）虚拟环境
  3. 切换虚拟环境

安装：

此时还不能使用virtualenvwrapper，默认virtualenvwrapper安装在/usr/local/bin下面，实际上你需要运行virtualenvwrapper.sh文件才行，先别急，打开这个文件看看,里面有安装步骤，我们照着操作把环境设置好。

创建目录用来存放虚拟环境

配置脚本文件

在 bashrc 文件中进行如下操作

workon/lsvirtualenv
mkvirtualenv [虚拟环境名称]
workon [虚拟环境名称]
rmvirtualenv [虚拟环境名称]
deactivate
