内部类的作用

2017-08-19 -Java PV:

一、 作用

  1. 内部类可以很好的实现隐藏，一般的非内部类，是不允许有 private 与protected权限的，但内部类可以加上这几个修饰词。

  2. 内部类拥有外围类的所有元素的访问权限，可以直接访问外部类的私有属性及方法

  3. 可以间接实现多重继承，多个内部类继承不同的类形成了一个类中复用了多个类的方法。

  4. 可以避免在继承一个父类和一个接口的时候导致的方法重复定义。

二、详解

1.实现隐藏

2.可以无条件地访问外围类的所有元素

3.可以间接实现多重继承
