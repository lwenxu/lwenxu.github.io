2017-08-09 -Java PV:

基本类型的对象包装，也就是将常用的基本数据类型包装成对象

  • byte Byte
  • short Short
  • int Integer
  • long Long
  • boolean Boolean
  • float Float
  • double Double
  • char Character 最常用的作用就是基本数据类型与字符串的转换

    1. 基本数据类型转字符串：

    基本数据类型+””
    基本数据类型类.toString(基本类型的数值)

    2.字符串转成基本数据类型：

    Integer.parseInt()
    Long.parseLog()
    对character不用转就是string

    3.进制转换：
