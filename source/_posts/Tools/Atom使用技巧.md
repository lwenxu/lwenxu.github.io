### 打开项目
```bash
ctrl + o
```
### 模糊查找文件
```bash
ctrl + t  / ctrl + p
ctrl + b 仅仅查找已经打开的文件
```
### 命令模式
```bash
ctrl + shift +p
```
### 另存为
```bash
ctrl + shift + s
```
### gmerge-conflicts
```bash
在使用 Git 进行合并和 rebase 的时候可以用 alt-m d 来激活这个插件，它会列出所有冲突的文件，将每一处冲突高亮，同时有按钮和快捷键供你快速选用某个版本，在你解决所有冲突后会提示你进行 Commit. 有了这个插件再也不同担心出冲突的时候看瞎眼了。
```
