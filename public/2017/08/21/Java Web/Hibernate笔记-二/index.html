2.一对多的级联操作

首先在一的一方的配置文件的set标签中写一个属性 cascade=”save-update,delete” 这个就是级联配置，然后我们在写具体的代码的时候我们只需要把多的一方设置到一的一方的对象set中就可了，此时直接操作了一的一方就能把多的一方顺便就操作了。
